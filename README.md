# Midterm-Itai1371-g7

## What this project is about
This group project uses the Telco Customer Churn dataset from Kaggle to prepare a clean dataset for machine learning.
The primary objective of this project is to predict whether a customer will churn based on customer demographics, services, contract type, payment method, tenure, and charges.

## Dataset
Original dataset URL: https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Project Steps
1. Load the raw dataset
2. Review missing values and data types
3. Split the dataset into 70% training and 30% testing
4. Perform EDA only on the training data
5. Handle missing values
6. One-hot encode categorical variables
7. Scale numerical variables
8. Save cleaned training and testing datasets


## Important Note
The original dataset was not changed manually. All preprocessing was performed in Python using Jupyter Notebook.
